A simple Number Guessing Game In Python

Welcome to the Number Guessing Game, a web project built with Python and Flask, complemented by HTML, CSS, and JavaScript. This interactive game allows users to submit their guesses for a mystery number. The Flask server handles the backend logic, while the front end provides a simple interface for user interaction. Whether you're interested in coding or just curious to try a guessing game, this project offers a blend of technology and entertainment. Feel free to explore and test your guessing skills
